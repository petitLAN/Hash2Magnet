About the .torrent download button

A magnet link can be made from just the hash because it only needs an xt=urn:btih: value. A real .torrent file contains extra metadata, so it cannot be created from only the hash in a fully offline/static page. This button therefore opens a public torrent-cache URL for 40-character v1 hashes. If the cache does not know the hash, there is no file to download.
